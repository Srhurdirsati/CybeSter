# CybeSter

CyberSter is about making a portable MP3 player using different electronic modules. It can play music from an SD card and send audio through AUX or Bluetooth. A screen is used to show the menu and song information. The device also uses a Type-C port for power and transferring music files.
I wanna build it for listening music most of tht time I travel because in India , most of the places where we travel are remote and don't have network also It's pathetic to download music andd store in a mobile .

I decided to have a Advanced MP3 player with A big enough display , sd card for input , bt support , Aux cable and Type C for charging and Data Transfer .

## Features

> A Big TFT Display for viewing contents and Song List

> Buttons For operating

> Good Audi oAmplifier

> SD card support

> Aux Support


NOTE - I have removed BT Beacause i made an error so I am currently using AUX output but will get future updates . 

## BOM 

 Main components:

> ESP32-S3 module

> VS1053 MP3 decoder

> SPI TFT display (ST7789 / similar)

> Micro SD card slot

> 6 push buttons (Play, Next, Prev, Volume Up, Volume Down, Menu)

> 3.5 mm headphone jack

> USB-C port for power

> TP4056 battery charger

> 3.7V Li-Po battery


## How To Upload

> Install Arduino IDE.

> Install the ESP32 board package.

> Install the required libraries:

> Adafruit VS1053

> Adafruit GFX

> Adafruit ST7789

> SPI

> SD

> Open the firmware file.

> Select your ESP32 board.

> Upload the code.
